## 🚀 About the Project
**Taapuu** is a scalable, feature-rich production-grade classified ads platform designed to provide a seamless marketplace experience for local communities. It allows users to post, browse, and manage advertisements completely for free, empowering peer-to-peer commerce and lowering the barrier to digital trade.

The platform is designed with a modern **dual-stack architecture**:
- **Frontend:** Built with **Flutter** for high-performance, cross-platform mobile experiences (iOS & Android).
- **Backend:** Powered by **Laravel**, providing a highly secure, optimized RESTful API framework for robust data handling and session management.

---

## 🛠️ Tech Stack & Architecture

### Frontend (Mobile App)
- **Framework:** Flutter (Dart) 🎯
- **State Management:** Clean Architecture with structured state handles.
- **UI Components:** Custom, highly reusable Widget Library tailored for fluid marketplace animations.
- **Networking:** Optimized HTTP client handling secure REST API integrations, token management, and interceptors.

### Backend & Infrastructure
- **Framework:** Laravel (PHP) 🐘
- **Database:** Relational Database with optimized indexing for rapid classified search queries.
- **Security:** Guarded endpoints with secure authentication and token-based validation layers.

---

## ✨ Key Features
- **Instant Ad Posting:** Multi-step fluid forms for uploading ad descriptions and high-resolution images.
- **Advanced Filtering & Search:** Category-based filters with real-time query optimization.
- **User Dashboard:** Comprehensive profile controls, listing management (active/inactive), and account statistics.
- **Interactive UI:** Smooth transitions, responsive grids, and customized native device interactions.

## 📦 Project Setup & Installation

To run the Flutter project locally for development or testing, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   cd tappuuApp
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the project:**
   ```bash
   flutter run
   ```
